# Security Policy

## Purpose

This repository is intended exclusively for educational purposes and professional portfolio development. It documents cybersecurity concepts, hands-on labs, Capture The Flag (CTF) challenges, and security research performed in authorized environments.

## Responsible Use

The techniques, tools, and methodologies presented in this repository must only be used:

* In personal laboratory environments.
* On intentionally vulnerable machines.
* During Capture The Flag competitions.
* On systems where explicit authorization has been granted.

Unauthorized access to computer systems is illegal and unethical.

## Reporting Issues

If you discover any accidental disclosure of sensitive information within this repository, please open an issue or contact the repository owner so it can be addressed promptly.

## Sensitive Information

This repository is regularly reviewed to ensure that it does not contain:

* Credentials
* API keys
* Private certificates
* Authentication tokens
* Personal information
* Proprietary or confidential data

If any such content is identified, it will be removed immediately.

## Disclaimer

The repository owner assumes no responsibility for any misuse of the information contained in this repository. Readers are solely responsible for ensuring their activities comply with applicable laws, regulations, and authorization requirements.
